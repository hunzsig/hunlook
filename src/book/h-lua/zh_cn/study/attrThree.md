## 三围属性影响

> 属性每1点属性映射 可以模拟三围加成等

```lua
hattr.setRelation({
    primary = 1,
    str = {
        life = 16,
        life_back = 0.03,
        aim = 0.003
    },
    agi = {
        attack_speed = 0.01,
        avoid = 0.003
    },
    int = {
        mana = 8,
        mana_back = 0.02
    }
})
```
