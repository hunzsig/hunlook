## 快速开始

###### 停止维护、完全开源、菜鸟入门

### 克隆（下载）项目

```
git clone https://github.com/h-lua/h-lua
```

> 不推荐使用zip下载，使用git能看到项目的开发历史，在fork的加持下也方便监听项目的更新
>
> 会git的自然就懂，而考虑很多人根本不会用git，这里直接演示fork的用法。

### 打开终端，进入项目并确保进入工作区

```
cd ./h-lua
```

### 创建新项目

```
> sdk.exe new demo
```

### 测试

```
> sdk.exe test demo
```
