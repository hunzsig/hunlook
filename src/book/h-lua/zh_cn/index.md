## 快速开始

* 版本稳定、安全安心
* 菜鸟入门、自由拓展
* 功能基础、门槛极低

### 获得框架

> https://github.com/h-lua/h-lua

### 打开终端，进入项目并确保进入工作区

```
cd ./h-lua
```

### 创建新项目

```
> sdk.exe new demo
```

### 测试

```
> sdk.exe test demo
```
