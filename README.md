# Hunlook

Hun book, a brief introduction and refreshing book writing project.

#### Start

```
git clone https://github.com/hunzsig/hunlook.git
cd hunlook
```

#### Install [nodejs npm](https://nodejs.org)

```
npm i
or
cnpm i
```

#### Test(Default Port 9971)

```
npm start
```

#### Package

```
npm run build
```
